# Natomas Local News Website

## Setup Instructions

Follow these steps to set up the Natomas local news website in your local environment:

1. **Clone the Repository**  
   Open your terminal and run the following command:
   ```bash
   git clone https://github.com/jaegerkyle/Natomas.git
   cd Natomas
   ```

2. **Install Dependencies**  
   Make sure you have [Node.js](https://nodejs.org/) installed. Then, run:
   ```bash
   npm install
   ```

3. **Running the Development Server**  
   To start the local development server, run:
   ```bash
   npm start
   ```
   You can access the website at `http://localhost:3000`.

4. **Build for Production**  
   To build the project for production, run:
   ```bash
   npm run build
   ```

5. **Testing**  
   To run tests, use the following command:
   ```bash
   npm test
   ```
